## Quick Start

### Prerequisites

Before starting make sure you have:

- Installed [Yarn](https://yarnpkg.com/)

### Installation

```bash
$ yarn install
```

### Starting Development

```bash
$ yarn start
```

### Building for Production

```
$ yarn build
```
