## 简介

[Flutter Widget Livebook](](https://flutter-widget.live)) 是一个使用 Flutter for web 构建的可在线实时预览 Widget 示例的网站。

## 模块

| 模块                                    | 描述                                        | &nbsp;                                                                                                       |

## 快速开始

### 必备条件

开始之前确保你已安装：

- 已安装 [Yarn](https://yarnpkg.com/)

### 安装

```bash
$ yarn
```

### 开始开发

```bash
$ yarn dev
```

### 构建生产

```
$ yarn build
```
